■概要，特徴  
主な機能：  
・設定情報一括取得機能，およびパス情報取得時の環境依存解消機能  
・ログ出力実装補助，ログ＆テキストの同時出力機能  
・各種のFrameworkオプション機能  
　　※資格情報一括取得，アウトプット毎フォルダ(YMDhmsフォルダ)自動生成，終了時実行結果メール自動送信，等々  
・[勝又自作アクティビティパック]　※数値toLetter変換，等々  
・[勝又自作ワークフローパーツ]　※営業日取得.xaml，クリックDL結果ファイル格納.xaml 等々
